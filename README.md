## ğŸš€ Getting Started

### Prerequisites
- Node.js 18+ 
- npm or yarn

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/AbdulMannan7860/portfolio-nextjs.git
   cd portfolio-nextjs
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run the development server**
   ```bash
   npm run dev
   ```

4. **Open your browser**
   Navigate to [http://localhost:3000](http://localhost:3000)

### Build for Production

```bash
npm run build
npm start
```

## ğŸ”§ Customization

### Updating Personal Information
1. Edit component files in `src/components/`
2. Update contact information in `Contact.jsx`
3. Modify project details in `Projects.jsx`
4. Update skills in `Skills.jsx`

### Styling Changes
1. Modify `src/app/globals.css` for global styles
2. Update Tailwind classes in components
3. Customize color scheme in CSS variables

### Adding New Sections
1. Create new component in `src/components/`
2. Import and add to `src/app/page.js`
3. Update navigation in `Navigation.jsx`
